atölye görselleridir. Web sayfasında kullanılan optimize edilmiş WebP kopyaları
`public/images/departments/electrical/workshops/` altında tutulur.

aktif olduğu anlamına gelmez. Aktif dal bilgisi için
`../bolumler-ve-dallari-tanitim.pdf` kaynak kabul edilmelidir.
